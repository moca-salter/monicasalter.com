About

I build websites that clients can maintain easily, users can understand quickly, and developers can modify seamlessly.

React is my framework of choice for building JavaScript web apps, but I've also dabbled in others like Vue and Angular. When I'm styling an app, I write modular, maintainable CSS, usually with Sass, and I love a well organized pattern library.

When I'm not buiding websites, I like to read, spend time outdoors, and play board games with friends!
